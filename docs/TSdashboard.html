About the dashboard

This dashboard is a result of an assignment performed for PEX820 - Data Manipulation and Visualization. This course was enrolled in 2022/2 semester and belongs to the Graduate Program in Statistics and Agricultural Experimentation from Universidade Federal de Lavras (UFLA).

The data present in this dashboard comes from Spotifys API (Spotify for Developers 2022) and was read into the R environment through the spotifyr (Thompson et al. 2021) package and manipulated using the tidyverse (Wickham et al. 2019) package. The Table presented on the next page showcases the data used from now on. We considered only the standard original recordings of all albums, although further analysis with the different editions and re-recordings are considered.

Other used packages: flexdashboard (Sievert et al. 2022), viridis (Garnier et al. 2021), ggiraph (Gohel and Skintzos 2022), reactable (Lin 2022), and reactablefmtr (Cuilla 2022).

You are able to download the datasets in Base form, as extract from the Spotify API, or in the Final form used in this dashboard.

Refer to the References page to view all sources of information and download the base and final dataset of this analysis.
